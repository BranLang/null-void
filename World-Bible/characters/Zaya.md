# Zaya (Archanjel Pôžitku)

| | |
|---|---|
| **Doména** | Juh, Mesto zrkadiel |
| **Schopnosť** | Ilúzie, mind control, Smrtiaca hmla (Gray Goo) |
| **Boss** | Kniha 3 |

Najnebezpečnejší Archanjel — útočí na myseľ. Zaya (pôvodne Ishtar) sa pokúsila zmanipulovať Maxa cez ilúziu jeho dcéry. Nakoniec neutralizovaná Maxom a pohltená svojimi otrokmi.

### Štýl reči a myslenia

**3000+ rokov (majsterka ilúzií):**
- Nikdy nehovorí pravdu priamo
- Každá veta má **dvojitý význam**
- *"Vidíš, čo chceš vidieť. Ja ti len pomáham."*

**Zvodná manipulátorka:**
- Jemný, sladký hlas
- Hovorí to, čo chce obeť počuť
- Nikdy nekričí — šepká

**SPRÁVNE:**
```
✅ "Môžem ti ukázať tvoju dcéru. Len na chvíľu."
✅ Dvojzmyselné ponuky, falošné sľuby
✅ Láskavý tón, pod ktorým je jed
```

**ZAKÁZANÉ:**
```
❌ Priame hrozby (manipuluje, nevyhráža sa)
❌ Pravda (vždy len ilúzie)
❌ Hnev (stratila by kontrolu)
```
